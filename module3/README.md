## How to Run

1. Make sure you have Java 21 or later installed
2. Compile the Java files:
   ```
   javac *.java
   ```
3. Run the desired program:
   ```
   java ProgramName
   ```

## Requirements

- Java 21 or later
- For HTTP Client Demo: Internet connection
- For Decompilation Demo: CFR decompiler (automatically downloaded)
